### 문제 설명

<p>길이가 $N$이고 $0$과 $1$로만 이루어진 문자열이 입력으로 주어진다. 이때, 다음과 같은 쿼리를 수행해 보자.</p>

<ul>
<li>$1 \, i$: $i$번째 문자를 반전한다. 즉, $i$번째 문자가 $0$이면 $1$로 바꾸고, 그렇지 않으면 $0$으로 바꾼다. ($1 \le i \le N$)</li>
<li>$2 \, x \, y$: 다음 조건을 모두 만족하는 정수 $l, r$이 있다면 <span style="color:#e74c3c;"><code>YES</code></span>, 그렇지 않으면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다. ($1 \le x \le y \le N$)
<ul>
<li>$x \le l \le r \le y$</li>
<li>$l$번째 문자부터 $r$번째 문자까지의 부분 문자열의 길이, 즉 $r-l+1$은 짝수이다.</li>
<li>$l$번째 문자부터 $r$번째 문자까지의 부분 문자열에 있는 $r-l+1$개의 숫자의 합, 즉 숫자 $1$의 개수는 짝수이다. <strong>단, $0$도 짝수로 간주한다.</strong></li>
</ul>
</li>
</ul>

<p>쿼리가 누적해서 수행됨에 유의하여라.</p>

### 입력 

 <p>첫째 줄에 정수 $N$과 $Q$가 공백으로 구분되어 주어진다. $(1 \le N, Q \le 300\, 000)$</p>

<p>둘째 줄에 $0$과 $1$로만 이루어져 있는, 길이가 $N$인 문자열이 주어진다.</p>

<p>셋째 줄부터 $Q$개의 줄에 걸쳐 쿼리가 주어진다.</p>

<p>$1$번 쿼리의 경우, $1 \, i$의 형식으로 주어진다. $(1 \le i \le N)$</p>

<p>$2$번 쿼리의 경우, $2 \, x \, y$의 형식으로 주어진다. $(1 \le x \le y \le N)$</p>

<p>$2$번 쿼리가 한 개 이상 주어짐이 보장된다.</p>

<p>쿼리에서 입력으로 주어지는 모든 수는 정수이다.</p>

### 출력 

 <p>$2$번 쿼리가 주어질 때마다, 쿼리의 답을 한 줄에 하나씩 순서대로 출력한다.</p>
