### 문제 설명

<p><a href="/problem/22940">선형 연립 방정식</a>을 마스터했으니 이번엔 선형 연립 부등식을 풀어보자.</p>

<p>일차 다항식과 상수가 부등호로 연결된 부등식을 선형 부등식이라 한다. 즉, 다음과 같은 식을 의미한다.</p>

<p style="text-align: center;">$Ax + B \le C$</p>

<p>선형 연립 부등식이란 유한개의 선형 부등식의 모임이다. 다음과 같은 부등식의 모임이 선형 연립 부등식의 예이다.</p>

<p style="text-align: center;">$\begin{cases} x + 6 \le 4 \\ -2x + 2 \le 10 \end{cases}$</p>

<p>선형 연립 부등식의 해란 선형 연립 부등식의 모든 부등식이 참이 되게 하는 미지수의 값이다. 선형 연립 부등식이 주어질 때 정수해의 개수를 출력해 보자.</p>

### 입력 

 <p>첫 번째 줄에 부등식의 개수 $N$이 주어진다. $(1 \le N \le 100\,000)$</p>

<p>두 번째 줄부터 $N$개의 줄에 걸쳐 한 줄에 하나씩 정수 $A$, $B$, $C$가 공백으로 구분되어 주어진다. $(-10^9 \le A,B,C \le 10^9$; $A\neq 0)$</p>

### 출력 

 <p>첫 번째 줄에 선형 연립 부등식의 정수해의 개수를 출력한다.</p>

<p>선형 연립 부등식의 정수해가 무한히 많은 경우 첫 번째 줄에 <code><span style="color:#e74c3c;">-1</span></code>을 대신 출력한다.</p>
