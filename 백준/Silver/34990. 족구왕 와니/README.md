### 문제 설명

<p>대전대신고에 진학한 와니는 공부를 위해 축구를 잠시 접었다. 공을 차는 것을 좋아했던 와니는 아쉬웠지만, 다행히 체육 시간에 족구 수업이 있다는 것을 알게 되었다.</p>

<p>수업 때 조금이라도 더 잘하기 위해 와니는 자신을 포함한 $N$명의 팀원들과 함께 로봇을 이용한 수비 연습을 하기로 했다.</p>

<p>다음은 연습 규칙이다.</p>

<ol>
<li>팀원들은 연습 내내 각자의 고정된 위치에 서 있다.</li>
<li>로봇은 총 $M$개의 공을 순서대로 미리 설정된 목표 좌표로 발사한다.</li>
<li>공이 목표 좌표에 도달하면, 다음과 같은 규칙으로 공을 처리한다.
<ul>
<li>자신의 고정 위치에서 목표 좌표까지의 직선거리가 자신의 최대 반경 이하인 팀원들을 모두 찾는다. 이러한 팀원만이 공을 찰 수 있다.</li>
<li>그러한 팀원을 찾을 수 없다면, 팀은 이 공을 놓친다. 이 경우 재정비를 위해 <strong>다음 차례의 공을 차지 않는다.</strong></li>
<li>그러한 팀원이 $1$명 이상 있다면, 조건을 만족하는 팀원들 중 실력 수치가 가장 높은 사람이 공을 찬다.</li>
</ul>
</li>
<li>공을 찬 후 다시 제자리로 이동한다. 즉, 모든 팀원은 공이 발사될 때마다 항상 같은 자리를 유지하고 있다.</li>
<li>공은 반드시 목표 좌표에 도달한 순간에만 찰 수 있으며, 공이 목표 좌표로 이동하는 중간에는 공을 찰 수 없다.</li>
</ol>

<p>각 팀원들에게는 움직일 수 있는 최대 반경과 실력이 주어지며, <strong>실력이 같은 팀원은 없음이 보장된다.</strong> 또한 로봇이 발사한 공은 항상 목표 지점에 정확히 떨어진다.</p>

<p>$M$개의 공이 모두 발사되었을 때, 와니는 공을 몇 번 찼는지 구해보자.</p>

### 입력 

 <p>첫번째 줄에 팀원의 수 $N$과 공의 개수 $M$이 공백으로 구분되어 주어진다. $(1 \leq N \leq 100;$ $1 \leq M \leq 1\,000)$</p>

<p>두번째 줄부터 $N$번에 걸쳐 $i$ 번째 팀원의 고정 위치 $X_i$, $Y_i$, 최대 반경 $R_i$, 실력 $S_i$가 공백으로 구분되어 주어진다. $(0 \leq X_i, Y_i, R_i \leq 10\,000;$ $1 \leq S_i \leq 100)$</p>

<p><strong>첫번째 팀원의 정보가 와니의 정보이며, 동일한 $X_i$, $Y_i$의 쌍은 주어지지 않는다.</strong></p>

<p>$N+2$번째 줄부터 $M$개의 줄에 걸쳐 $k$번째 공의 목표 좌표 $TX_k$, $TY_k$가 공백으로 구분되어 주어진다. $(0 \leq TX_k, TY_k \leq 10\,000)$</p>

<p>입력으로 주어지는 수는 모두 정수이다.</p>

### 출력 

 <p>첫번째 줄에 와니가 공을 몇 번 찼는지 출력한다.</p>
